操作环境：WIN10
自用，待分类整理

一、下载、配置
1.打开powershell，输入winget install ffmpeg
2.重启powershell并敲ffmpeg查看是否有刷出相关信息；
若没有。复制ffmpeg所在目录，添加到系统环境变量中。
二、基础命令
1.FFMpeg
用于操作、转换和流式传输多媒体内容
2.FFprobe
媒体参数分析工具
3.FFplay
剪辑的播放器
三、转换格式
示例1：ffmpeg -i input.mov output.mp4
-i 为input的意思
-c 为codec，编解码器的意思

·想维持源视频文件的质量，可使用-qscale 0 参数：

ffmpeg -i input.webm -qscale 0 output.mp4

检查FFmepg的支持格式列表：
ffmpeg -formats

示例2：ffmpeg -i input.mp4 -vn output.mp3
将mp4视频文件转换成MP3音频文件

示例3：ffmpeg -i input.mp4 -vn -ar 44100 -ac 2 -ab 320 -f mp3 output.mp3
-vn 表示已经在输出文件中禁用视频录制
-ar 设置输出文件的音频频率。通常使用的值是22050HZ、44100HZ、48000HZ
-ac 设置音频通道的数目
-ab 表明音频比特率
-f 输出文件格式

示例4：ffmpeg -i input.mp4 -filter:v scale=1280:720 -c:a copy output.mp4
或 ffmpeg -i input.mp4 -s 1280 * 720 -c:a copy output.mp4
将所给定视频文件的分辨率到1280 * 720

示例5：ffmpeg -i input.mp4 -vf scale=1280:-1 -c:v libx264 -preset veryslow -crf 24 output.mp4
压缩并减小输出文件的大小

也可以通过下面的选项来转换编码音频降低比特率，使其更有立体声感，从而减小大小；
-ac 2 -c :a aac -strict -2 -b:a 128k

示例6：ffmpeg -i input.mp3 -ab 128 output.mp3
-ab 标志压缩音频文件
各种可用的音频比特率列表是：
96kbps
112kbps
128kbps
160kbps
192kbps
256kbps
320kbps

示例7：ffmpeg -i input.mp4 -an output.mp4
-an 从一个视频文件移除音频流
-vn 同理，表示从媒体文件移除视频流
如 ffmpeg -i input.mp4 -vn output.mp3

示例8：ffmpeg -i input.mp4 -r 1 -f image2 image-%2d.png
从视频中提取图像
-r 设置帧速度。即每秒提取帧到图像的数字。默认值是25
-f 表示输出格式，如图像
image-%2d.png 表明想如何命名提取的图像
